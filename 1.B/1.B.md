# Exercise 1.B: Stock Correlation Matrix Tool

## Purpose

This exercise is designed to deepen your proficiency in **numerical computing and statistical analysis** for finance. You will learn to:
- Use `numpy` and `pandas` for advanced data manipulation
- Calculate daily returns to measure stock performance over time
- Compute and interpret a correlation matrix for multiple financial assets
- Discover sectoral relationships and market dynamics using heatmaps
- Gain experience handling large-scale datasets with multiple files

This exercise bridges data science fundamentals with portfolio management, demonstrating how diversification and sector analysis are driven by quantitative methods.

---

## Objective

Build a Python tool that reads historical data for 50 different stocks, calculates their daily returns, and uses NumPy to generate a **50x50 correlation matrix** visualized as a heatmap.

---

## Requirements

### Input
- Access 50 different stock CSV files from the [Full US Stock Market Dataset](https://www.kaggle.com/datasets/borismarjanovic/price-volume-data-for-all-us-stocks-etfs)
- **Recommended Tickers (5 Sectors x 10 Stocks):**
  - **Technology:** AAPL, MSFT, GOOGL, NVDA, META, AVGO, CSCO, ORCL, ADBE, CRM
  - **Finance:** JPM, BAC, WFC, GS, MS, AXAX, C, BLK, SCHW, PYPL
  - **Energy:** XOM, CVX, COP, SLB, EOG, MPC, PSX, VLO, HES, OXY
  - **Healthcare:** JNJ, UNH, LLY, ABBV, MRK, PFE, TMO, ABT, AMGN, DHR
  - **Consumer Goods:** AMZN, WMT, KO, PEP, PG, COST, MCD, NKE, PM, HD
- (Optionally) Accept a list of tickers or a directory path as user input

### Processing
1. **Data Loading**: Load historical price data (Close or Adjusted Close) for all 50 selected stocks
2. **Market Alignment**: Align all stock data by date; handle missing dates using interpolation or deletion to ensure consistency
3. **Returns Calculation**: Compute daily percentage returns for each stock: $R_t = \frac{Price_t - Price_{t-1}}{Price_{t-1}}$
4. **Correlation Matrix**: Use `numpy.corrcoef` or `pandas.corr()` to compute the $50 \times 50$ correlation matrix
5. **Insights**: Identify the most and least correlated pairs of stocks

### Output
Create a visualization that displays:
1. **Heatmap**: A $50 \times 50$ grid reflecting correlation strength using a color gradient (e.g., color map 'coolwarm' or 'RdBu')
2. **Labels**: Ticker symbols on both the X and Y axes
3. **Color Bar**: A legend indicating the correlation coefficient range (-1 to 1)
4. **Title**: Clearly state the analysis period and purpose (e.g., "S&P 500 Sector Correlation Heatmap (2020-2024)")
5. **File Export**: Save the resulting plot as a high-resolution PNG or JPG file

## Hints & Tips

1. **Efficient Loading**: Use `glob.glob('data/*.csv')` to quickly retrieve multiple filenames.
2. **Returns Formula**: `df.pct_change()` is the easiest way to compute daily returns in Pandas.
3. **Seaborn Styling**: Use `sns.heatmap(data, annot=False, cmap='coolwarm')` for a clean, visual representation of 50x50 matrices.
4. **Financial Logic**: Stocks in the same sector (e.g., Tech, Energy) typically show higher positive correlation (>0.7), while unrelated sectors or hedged assets show low or negative correlation.
